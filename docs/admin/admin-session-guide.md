# Admin Session Guide

## Session Monitoring
Admin > Sessions shows active and historical sessions.

### Columns
- Session ID
- From device (viewer)
- To device (host)
- Status (Pending/Active/Closed/Rejected)
- Start time
- Duration
- Data transferred

### Real-time Monitoring
- Active session count
- Average session duration
- Connection quality metrics
- Geographic distribution

### Actions
**View Details**: Participants, duration, quality
**Terminate**: Force end active session
**Export Logs**: Download session metadata

### Historical Analysis
- Session trends (daily/weekly/monthly)
- Peak usage times
- Common device pairs
- Failure reasons

## Alerts
Configure alerts for:
- Unusual session duration (> 8 hours)
- High failure rate
- Concurrent session spikes
