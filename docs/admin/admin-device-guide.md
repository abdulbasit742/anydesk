# Admin Device Guide

## Device Overview
Admin > Devices shows all registered devices.

### Device Information
- RemoteDesk ID (9 digits)
- Device name, OS, version
- Status (Online/Offline/Busy)
- Owner
- Last seen timestamp

### Actions
**View Details**: Click device row
**Disconnect**: Force disconnect from active session
**Remove**: Unregister device (requires confirmation)
**Transfer**: Move to different organization

### Filtering
- By status: Online/Offline/Busy
- By OS: Windows/macOS/Linux
- By organization
- By owner

### Bulk Actions
- Select multiple devices
- Bulk disconnect
- Bulk remove
- Export device list

## Device Policies
Admins can configure device-level policies:
- Allowed OS versions
- Required security settings
- Session timeout
- Unattended access restrictions
