# Admin User Guide

## User Management

### Viewing Users
Navigate to Admin > Users to see all registered users.
- Search by email or name
- Filter by role, status, organization
- Export user list to CSV

### Suspending Users
1. Find user in user list
2. Click "Suspend" button
3. Confirm suspension reason
4. User is immediately logged out
5. Audit log entry created

### Reactivating Users
1. Find suspended user
2. Click "Reactivate"
3. User can log in again

### Changing User Roles
1. Find user
2. Click "Edit Role"
3. Select new role (USER, ADMIN, SUPPORT)
4. Confirm change

## Audit Trail
All admin actions are logged in the audit log.
Access: Admin > Audit Logs

## Security Notes
- Admin actions require re-authentication for sensitive operations
- Bulk operations require additional confirmation
- All actions are irreversible (except reactivation)
