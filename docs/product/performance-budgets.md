# Performance Budgets

## API
- P50 response time: < 50ms
- P95 response time: < 200ms
- P99 response time: < 500ms
- Error rate: < 0.1%

## Web
- First Contentful Paint: < 1.5s
- Time to Interactive: < 3s
- Bundle size: < 200KB (initial)

## Desktop
- App startup: < 3s
- Session connection: < 5s
- Screen capture latency: < 100ms

## WebRTC
- ICE gathering: < 3s
- Connection establishment: < 5s
- Frame rate: > 15 FPS (minimum)
- End-to-end latency: < 200ms

## Database
- Query time: < 10ms (simple), < 100ms (complex)
- Connection pool: 10-50 connections

## Infrastructure
- CPU: < 70% average
- Memory: < 80% average
- Disk: < 85% used
