# RemoteDesk Brand Guidelines

## Logo
- Primary: Blue gradient (#0066FF to #00CCFF)
- Monochrome: Dark gray (#1A1A2E)
- Minimum size: 32px height
- Clear space: Equal to logo height

## Colors
- Primary: #0066FF
- Secondary: #00CCFF
- Success: #00C853
- Warning: #FFB300
- Danger: #FF1744
- Background: #F5F7FA
- Surface: #FFFFFF
- Text: #1A1A2E

## Typography
- Headings: Inter, 600 weight
- Body: Inter, 400 weight
- Mono: JetBrains Mono (for IDs, code)

## Voice
- Professional but approachable
- Clear and concise
- Security-focused
- Helpful and supportive

## Usage
- "RemoteDesk" (one word, camel case)
- "RemoteDesk ID" (not "remotedesk id")
- Feature names: sentence case
- Buttons: verb + noun (e.g., "Start Session")
