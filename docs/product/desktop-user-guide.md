# Desktop User Guide

## Installation

### macOS
1. Download `.dmg` from [remotedesk.io/download](https://remotedesk.io/download)
2. Open `.dmg` and drag to Applications
3. Allow screen recording permission when prompted

### Windows
1. Download `.exe` installer
2. Run installer and follow prompts
3. Allow firewall exception

### Linux
1. Download `.AppImage`
2. Make executable: `chmod +x RemoteDesk-*.AppImage`
3. Run: `./RemoteDesk-*.AppImage`

## First Launch
1. Sign in or create account
2. Grant screen recording permission
3. Your device gets a 9-digit RemoteDesk ID

## Connecting to a Remote Device
1. Enter the RemoteDesk ID in the input field
2. Click "Connect"
3. Wait for host to accept
4. Session starts automatically

## During a Session
- View remote screen in real-time
- Request remote input permission if needed
- Share clipboard (if permitted)
- Transfer files (if permitted)
- Adjust quality settings

## Hosting a Session
1. Keep RemoteDesk running
2. Your ID is visible on the main screen
3. When someone connects, accept or reject
4. Control what they can do via permissions

## Settings
- **General**: Startup options, theme
- **Security**: 2FA, trusted devices, password
- **Permissions**: Default permission settings
- **Network**: Proxy settings, ICE servers
- **Quality**: Bandwidth limits, resolution

## Troubleshooting
- **Can't connect**: Check internet, verify ID
- **Poor quality**: Reduce resolution, check bandwidth
- **Black screen**: Grant screen recording permission
- **Disconnected**: Check firewall, try relay mode

## Keyboard Shortcuts
- `Ctrl/Cmd + Shift + C`: Copy
- `Ctrl/Cmd + Shift + V`: Paste
- `Ctrl/Cmd + F`: Fullscreen
- `Esc`: Exit fullscreen
- `Ctrl/Cmd + Q`: Quit
