# Support Handbook

## Ticket Categories
- **Technical**: Connection issues, bugs, crashes
- **Billing**: Payments, subscriptions, refunds
- **Account**: Registration, login, profile
- **Feature Request**: New feature suggestions
- **Bug**: Confirmed software defects

## Priority Levels
- **Urgent**: Service outage, data loss
- **High**: Major feature broken
- **Medium**: Partial functionality issue
- **Low**: Cosmetic, feature request

## Response Times
- Urgent: 1 hour
- High: 4 hours
- Medium: 24 hours
- Low: 72 hours

## Escalation
1. L1 Support: Initial triage
2. L2 Support: Technical investigation
3. Engineering: Bug fixes
4. Management: Escalations, refunds

## Common Issues
- **Can't connect**: Check ID, firewall, internet
- **Black screen**: Permissions, capture source
- **Slow performance**: Bandwidth, quality settings
- **Login issues**: Password reset, 2FA
- **Billing**: Card update, invoice questions

## Tools
- Admin dashboard for user lookup
- Session logs for debugging
- Audit logs for account issues
- Stripe dashboard for billing
