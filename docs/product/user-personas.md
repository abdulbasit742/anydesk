# User Personas

## IT Administrator: Alex
- **Role**: IT Admin at mid-size company
- **Goals**: Manage remote access for team, enforce security policies
- **Pain Points**: Managing many devices, ensuring compliance
- **Uses**: Admin dashboard, policies, audit logs

## Remote Worker: Sarah
- **Role**: Software developer working from home
- **Goals**: Access office workstation, collaborate with team
- **Pain Points**: Connection reliability, ease of use
- **Uses**: Desktop app, session management

## Support Technician: Mike
- **Role**: Customer support at SaaS company
- **Goals**: Quickly access customer systems to troubleshoot
- **Pain Points**: Waiting for customer approval, multiple sessions
- **Uses**: Web app, support tickets, session history

## Enterprise Manager: Linda
- **Role**: CISO at enterprise company
- **Goals**: Ensure security compliance, manage access
- **Pain Points**: Compliance requirements, audit trails
- **Uses**: Policies, audit logs, user management

## Casual User: Tom
- **Role**: Help family with tech issues
- **Goals**: Simple, free remote help
- **Pain Points**: Technical complexity, cost
- **Uses**: Free tier, web app
