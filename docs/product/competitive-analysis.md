# Competitive Analysis

## AnyDesk
- **Strengths**: Fast, lightweight, good performance
- **Weaknesses**: Limited collaboration features, security concerns
- **Differentiator**: RemoteDesk has better security and enterprise features

## TeamViewer
- **Strengths**: Full-featured, established brand
- **Weaknesses**: Expensive, complex UI
- **Differentiator**: RemoteDesk is simpler and more affordable

## Chrome Remote Desktop
- **Strengths**: Free, easy setup
- **Weaknesses**: Requires Google account, limited features
- **Differentiator**: RemoteDesk has native apps and enterprise features

## RustDesk
- **Strengths**: Open source, self-hosted option
- **Weaknesses**: Less polished, smaller community
- **Differentiator**: RemoteDesk has better SaaS offering and support

## Parsec
- ** **Strengths**: Low latency, gaming-focused
- **Weaknesses**: Limited business features
- **Differentiator**: RemoteDesk is business-focused with compliance

## RemoteDesk Positioning
- Best-in-class security
- Simple 9-digit ID system
- Enterprise policy enforcement
- Web-first with native apps
- Transparent pricing
