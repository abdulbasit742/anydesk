# Glossary

**RemoteDesk ID**: A 9-digit numeric identifier assigned to each device for easy connection.

**Host**: The device being remotely accessed.

**Client**: The device initiating the remote connection.

**Session**: An active remote desktop connection between two devices.

**WebRTC**: Web Real-Time Communication, the protocol used for peer-to-peer connections.

**ICE**: Interactive Connectivity Establishment, the process of finding the best connection path.

**TURN Server**: A relay server used when direct peer-to-peer connection is not possible.

**Signaling**: The process of exchanging connection information between peers via the server.

**SDP**: Session Description Protocol, used to describe media capabilities.

**NAT Traversal**: Techniques used to establish connections through network address translators.

**2FA**: Two-Factor Authentication, requiring a second verification step.

**TOTP**: Time-based One-Time Password, a 2FA method using time-synchronized codes.

**RBAC**: Role-Based Access Control, permissions based on user roles.

**GDPR**: General Data Protection Regulation, EU data privacy law.

**Proration**: Adjusting subscription charges when changing plans mid-cycle.

**Coturn**: Open-source TURN and STUN server implementation.
