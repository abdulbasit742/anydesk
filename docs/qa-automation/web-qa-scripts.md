- Chrome 120+

2. Desk ID displayed prominently
3. Connection status indicator works

5. Dark/light mode toggle works

4. Mobile (375x812) - single column

2. Screen reader announces states
3. Focus indicators visible
4. Color contrast meets WCAG AA
5. Alt text on all images
