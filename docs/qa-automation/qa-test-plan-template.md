# QA Test Plan Template

## 1. Scope
### In Scope
- Feature X functionality
- API endpoints
- UI/UX flow

### Out of Scope
- Performance testing
- Security testing
- Load testing

## 2. Test Types
| Type | Count | Approach |
|------|-------|----------|
| Unit | N | Automated |
| Integration | N | Automated |
| E2E | N | Automated + Manual |
| Exploratory | N | Manual |
| Accessibility | N | Automated + Manual |

## 3. Entry Criteria
- Code complete
- Unit tests passing
- Code review approved
- Deployed to staging

## 4. Exit Criteria
- All P0 tests pass
- < 5% failure rate
- No open critical bugs
- Performance within SLA
- QA sign-off obtained

## 5. Risks
| Risk | Mitigation |
|------|-----------|
| Time constraint | Prioritize P0 tests |
| Environment issues | Have backup environments |
