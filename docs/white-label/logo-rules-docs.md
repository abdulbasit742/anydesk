# RemoteDesk Logo Rules

## Requirements

### Format
| Format | Use Case | Notes |
|--------|----------|-------|
| SVG | Web, Desktop | Preferred, scalable |
| PNG | Fallback | Min 2x resolution |
| ICO | Desktop app | Windows taskbar |
| ICNS | Desktop app | macOS dock |

### Dimensions
| Context | Recommended Size |
|---------|-----------------|
| Header | Height: 40px |
| Favicon | 32x32px |
| App icon | 512x512px |
| Loading | 120x120px |
| Email | Height: 60px |

### Safe Area
Maintain 20% padding around logo content:
```
+----------------------------+
|        20% padding         |
|    +------------------+    |
|    |                  |    |
|    |     LOGO         |    |
|    |                  |    |
|    +------------------+    |
|        20% padding         |
+----------------------------+
```

### Contrast
- Light background: Use dark logo variant
- Dark background: Use light/inverted logo variant
- Minimum contrast ratio: 4.5:1

### Prohibited
- Stretching or distorting
- Changing brand colors
- Adding effects (shadows, glows)
- Using on busy backgrounds
- Rotating or tilting

## Upload Process
1. Admin Dashboard -> Branding -> Logo
2. Upload SVG or PNG
3. Preview on light and dark backgrounds
4. Adjust if needed
5. Save changes

## Validation
- File size < 500KB
- Valid image format
- Minimum 200x200px
- Not offensive content
