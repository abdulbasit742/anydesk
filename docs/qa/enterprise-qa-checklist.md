# Enterprise QA Checklist

## Organization
- [ ] Create organization
- [ ] Update organization
- [ ] Delete organization (with confirmation)
- [ ] Slug validation

## Members
- [ ] Invite by email
- [ ] Invite link
- [ ] Accept invitation
- [ ] Decline invitation
- [ ] Remove member
- [ ] Transfer ownership

## Roles
- [ ] Owner has full access
- [ ] Admin can invite/manage
- [ ] Member can view/use
- [ ] Role changes take effect

## Policies
- [ ] Create device policy
- [ ] Create session policy
- [ ] Create security policy

- [ ] Policy enforcement works
- [ ] Disable policy stops enforcement

## Audit
- [ ] All actions logged
- [ ] Logs filterable
- [ ] Logs exportable
- [ ] Log tamper-evident

## Billing
- [ ] Enterprise plan features
- [ ] Custom pricing
- [ ] Invoice generation
- [ ] PO support
