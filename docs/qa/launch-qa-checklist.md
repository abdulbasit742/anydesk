# Launch QA Checklist

## Pre-Launch
- [ ] All automated tests passing
- [ ] Security review completed
- [ ] Performance benchmarks met
- [ ] Accessibility audit passed
- [ ] Cross-browser testing done
- [ ] Mobile responsive verified

## Critical Paths
- [ ] User registration
- [ ] User login/logout
- [ ] Password reset
- [ ] Device registration
- [ ] Session establishment (host)
- [ ] Session establishment (viewer)
- [ ] Session termination
- [ ] Screen sharing quality
- [ ] Remote control
- [ ] Chat during session
- [ ] File transfer
- [ ] Clipboard sync

## Billing
- [ ] Plan display correct
- [ ] Checkout flow works
- [ ] Trial activation
- [ ] Subscription renewal
- [ ] Cancellation flow
- [ ] Invoice generation

## Admin
- [ ] User management
- [ ] Device management
- [ ] Session monitoring
- [ ] Audit log access
- [ ] Billing management

## Post-Launch
- [ ] Error rates < 0.1%
- [ ] Response times < 500ms (p95)
- [ ] Support tickets < 10/day
- [ ] No critical bugs
- [ ] Customer satisfaction > 4.0/5
