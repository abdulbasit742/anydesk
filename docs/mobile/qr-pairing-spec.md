# QR Code Pairing Specification

## Purpose
Allow quick pairing between mobile and desktop without typing 9-digit ID.

## Flow
1. Desktop generates QR code containing RemoteDesk ID
2. Mobile scans QR code
3. Mobile auto-populates ID and initiates connection
4. Desktop accepts (or auto-accepts if trusted)

## QR Code Format
```
remotedesk://connect?id=123456789&name=Desktop-PC
```

## QR Code Standards
- Version: 4 (33x33 modules)
- Error correction: M (15%)
- Encoding: Alphanumeric

## Security
- QR codes expire after 5 minutes
- One-time use
- Must be generated by authenticated device
- Visual indicator when QR is active

## API
```
POST /devices/{id}/qr-code
-> { qrCodeUrl: "data:image/png;base64,...", expiresAt: "2024-01-01T00:05:00Z" }
```

## Mobile Scanning
- Use native camera APIs
- ZXing (Android) / AVFoundation (iOS)
- Validate scheme is `remotedesk://`
- Extract ID parameter
