# Common Issues Guide

## Connection Issues

### "Device Offline"
**Symptoms**: Cannot connect, shows offline
**Causes**:
- Desktop client not running
- Network issue
- Firewall blocking

**Resolution**:
1. Confirm desktop client is open
2. Check internet connection
3. Verify firewall allows RemoteDesk
4. Restart desktop client
5. Check status page

### "Connection Timeout"
**Symptoms**: Request sent but no response
**Causes**:
- Host not accepting
- Network latency high
- TURN server issue

**Resolution**:
1. Ask host to check for request
2. Try again in 1 minute
3. Check both have stable internet
4. Contact support if persistent

## Authentication Issues

### "Invalid Credentials"
1. Check Caps Lock
2. Try password reset
3. Verify email address
4. Check account not suspended

### "Session Expired"
1. Re-authenticate
2. Check browser cookies enabled
3. Clear cache and retry

## Performance Issues

### "Laggy Connection"
1. Check internet speed (need 1Mbps+)
2. Reduce quality in settings
3. Close bandwidth-heavy apps
4. Try different network

### "Low Frame Rate"
1. Reduce screen resolution
2. Lower quality setting
3. Check CPU usage on host
4. Close unnecessary applications
