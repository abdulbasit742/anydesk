# Support Ticketing System

## Overview
Built-in support ticket management for RemoteDesk customers.

## Ticket Lifecycle
1. **Open**: New ticket created
2. **In Progress**: Assigned agent working
3. **Waiting**: Awaiting customer response
4. **Resolved**: Issue resolved
5. **Closed**: Ticket closed (customer can reopen)

## Categories
- Technical, Billing, Account, Feature Request, Bug

## Internal Notes
- Only visible to admin/support staff
- Audit logged separately
- Not included in customer exports

## User Timeline
Aggregated view of user's tickets, sessions, and devices for support context.
