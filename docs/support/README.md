# Customer Support

## Ticket Workflow
- open -> in-progress (agent)
- in-progress -> resolved (any)
- in-progress -> escalated (agent)
- resolved -> closed (any)
- resolved -> open (user, reopen)

## SLA
- Urgent: 2 hours
- High: 8 hours
- Normal: 24 hours
- Low: 72 hours

## Components
- SupportDashboardPage - Overview stats
- SupportNewTicketPage - Ticket creation form
- SupportTicketsPage - Ticket list with filters
- SupportTicketDetailPage - Ticket view + comments
- InternalNotesPanel - Agent-only notes
- UserTimeline - User activity history
- SlaIndicator - SLA compliance display

## Pages
- /support - Dashboard
- /support/new - Create ticket
- /support/tickets - My tickets
- /support/tickets/:id - Ticket detail
- /support/kb - Knowledge base
- /support/contact - Contact form
