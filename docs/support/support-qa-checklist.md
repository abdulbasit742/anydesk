# Support QA Checklist

## Process QA
- [ ] Ticket created within 5 minutes of contact
- [ ] Auto-response sent immediately
- [ ] SLA timer starts on creation
- [ ] Escalation rules trigger correctly
- [ ] Customer notified on resolution

## Template QA
- [ ] All templates have current branding
- [ ] Variables populate correctly
- [ ] Links are valid
- [ ] Tone is consistent

## Knowledge Base QA
- [ ] All articles under 6 months old
- [ ] Screenshots match current UI
- [ ] Steps are accurate
- [ ] No broken links

## Tools QA
- [ ] Ticketing system accessible
- [ ] Search works
- [ ] Filters work
- [ ] Export works
- [ ] Metrics dashboard loads

## Training QA
- [ ] New agents complete training
- [ ] Agents can access all resources
- [ ] Escalation procedures known
- [ ] SLA targets understood
