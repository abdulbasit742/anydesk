# GDPR Compliance

## User Rights
- **Access**: Users can export all their data via `/compliance/export`
- **Erasure**: Users can request account deletion via `/compliance/deletion`
- **Portability**: Export is in standard JSON format

## Implementation
- 14-day grace period for deletion requests
- All actions audit logged
- Confirmation required before deletion

## Data Processing Agreement
Available upon request for Enterprise customers.
