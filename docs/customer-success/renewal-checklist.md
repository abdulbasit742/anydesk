# RemoteDesk Renewal Checklist

## 120 Days Before Renewal
- [ ] Review account health score
- [ ] Analyze usage trends
- [ ] Identify expansion opportunities
- [ ] Schedule business review
- [ ] Review support history

## 90 Days Before Renewal
- [ ] Conduct QBR
- [ ] Present ROI analysis
- [ ] Address any concerns
- [ ] Introduce new features
- [ ] Begin pricing discussion

## 60 Days Before Renewal
- [ ] Send renewal proposal
- [ ] Negotiate terms
- [ ] Get verbal commitment
- [ ] Prepare contract
- [ ] Identify upsell opportunities

## 30 Days Before Renewal
- [ ] Contract signed
- [ ] Payment processed
- [ ] Thank you sent
- [ ] Expansion opportunities followed up
- [ ] Case study requested

## At Risk Process
If renewal at risk:
- [ ] Escalate to manager
- [ ] Offer discount (up to 15%)
- [ ] Propose flexible terms
- [ ] Executive involvement
- [ ] Retention offer
