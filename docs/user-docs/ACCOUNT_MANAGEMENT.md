# Account Management

## Profile
### Update Name
1. Go to Settings > Profile
2. Edit your name
3. Click Save

### Change Avatar
1. Go to Settings > Profile
2. Click avatar
3. Upload new image
4. Crop and save

### Change Email
1. Go to Settings > Profile
2. Click "Change Email"
3. Enter new email
4. Verify via link

## Security
### Change Password
1. Go to Settings > Security
2. Click "Change Password"
3. Enter current password
4. Enter new password

### Enable 2FA
See Security Guide for detailed steps.

### Manage Sessions
1. Go to Settings > Security > Sessions
2. Review active sessions
3. Revoke any unknown sessions

## Deletion
### Delete Account
1. Go to Settings > Account
2. Click "Delete Account"
3. Confirm with password
4. All data will be deleted

### Export Data
1. Go to Settings > Privacy
2. Click "Export My Data"
3. Download will be emailed

## Organization
### Leave Organization
1. Go to Organization > Settings
2. Click "Leave Organization"
3. Confirm (cannot be undone)

### Transfer Ownership
1. Go to Organization > Members
2. Click "Transfer Ownership"
3. Select new owner
4. Verify with code
