# New User Onboarding Checklist

## Account Setup
- [ ] Create account
- [ ] Verify email
- [ ] Set up 2FA
- [ ] Complete profile

## First Connection
- [ ] Install RemoteDesk
- [ ] Find your RemoteDesk ID
- [ ] Connect to a test device
- [ ] Try file transfer

## Team Setup (Admin)
- [ ] Create organization
- [ ] Invite team members
- [ ] Set roles
- [ ] Configure security

## Advanced
- [ ] Try session recording
- [ ] Set up keyboard shortcuts
- [ ] Explore settings
- [ ] Read security guide

## Support
- [ ] Bookmark documentation
- [ ] Join community forum
- [ ] Know how to submit tickets
- [ ] Check status page
