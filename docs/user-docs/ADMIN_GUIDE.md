# RemoteDesk Admin Guide

## Dashboard
The admin dashboard provides an overview of your organization.

### Stats Cards
- **Total Users**: Members in your organization
- **Active Devices**: Currently online devices
- **Active Sessions**: Ongoing remote sessions
- **Alerts**: Issues requiring attention

## Managing Users
### Inviting Users
1. Go to Organization > Members
2. Click "Invite User"
3. Enter email and select role
4. Send invitation

### Roles
- **Owner**: Full control, cannot be removed
- **Admin**: Can manage users, view sessions
- **Member**: Basic access

### Deactivating Users
1. Find the user in the members list
2. Click the menu and select "Deactivate"
3. Confirm the action

## Managing Devices
- View all devices in your organization
- Filter by type, OS, status
- Revoke device access if needed

## Session Monitoring
- View active and historical sessions
- See participants and duration
- Check if recording is enabled

## Audit Log
- View all security events
- Filter by action type, result
- Export for compliance
- Track: logins, connections, permissions changes

## Organization Settings
### Security
- Require 2FA for admins
- Session timeout settings
- Auto-lock configuration

### Billing
- View current plan
- Manage subscription
- View invoices
- Update payment method

## Best Practices
- Regularly review audit logs
- Remove inactive users
- Keep software updated
- Train users on security
- Use strong permission controls
