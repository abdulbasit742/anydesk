# RemoteDesk Enterprise Guide

## Getting Started
### Deployment Options
- Cloud-hosted (SaaS)
- Private cloud
- On-premises

### SSO Setup
1. Go to Organization > Security
2. Click "Configure SSO"
3. Enter SAML/OIDC details
4. Test configuration
5. Enable for all users

### User Provisioning
- SCIM supported
- Auto-provision from IdP
- Role mapping available
