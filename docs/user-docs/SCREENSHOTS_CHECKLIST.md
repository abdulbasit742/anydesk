# Screenshots Checklist

## Marketing Screenshots
- [ ] Hero/main screen
- [ ] Connection in progress
- [ ] Active session (high quality)
- [ ] File transfer
- [ ] Mobile/responsive view

## Documentation Screenshots
- [ ] Dashboard overview
- [ ] User management
- [ ] Device list
- [ ] Session history
- [ ] Billing page
- [ ] Security settings
- [ ] Support ticket form

## Tutorial Screenshots
- [ ] Installation steps (each OS)
- [ ] First connection
- [ ] Permission request
- [ ] Quality settings
- [ ] Disconnect flow

## Contextual Help Images
- [ ] Empty states
- [ ] Error states
- [ ] Loading states
- [ ] Success confirmations
- [ ] Warning dialogs

## Screenshot Guidelines
- Use 1920x1080 resolution
- Clean browser (no extensions visible)
- Consistent test data
- Annotate with arrows/labels
- Dark and light theme variants
- English language UI
