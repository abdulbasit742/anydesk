# RemoteDesk Mobile Guide

## Current Status
Native mobile apps are in development.

## Web App on Mobile
You can use the web app on mobile browsers:
1. Open https://app.remotedesk.io
2. Sign in
3. Enter a RemoteDesk ID
4. View-only mode on mobile

## Tips for Mobile Viewing
- Use landscape orientation
- Pinch to zoom
- Double-tap for details
- Use on-screen controls
