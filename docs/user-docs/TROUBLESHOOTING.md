# Troubleshooting Guide

## Connection Issues

### "Host is busy"
- The remote device is in another session
- Wait and try again
- Contact the host

### "Host is offline"
- The remote device is not connected
- Check with the host
- Verify their RemoteDesk ID

### "Connection timeout"
- Check your internet connection
- Try a different network
- Check firewall settings
- Restart the app

### "ICE negotiation failed"
- Firewall blocking UDP
- Try enabling TURN relay
- Check NAT type
- Use a different network

## Video Issues

### Black screen
- Check screen capture permissions
- Try a different source
- Restart the app
- Update graphics drivers

### Low quality
- Check bandwidth
- Lower quality settings
- Close bandwidth-heavy apps
- Use wired connection

### Lag/stuttering
- Reduce quality to Low
- Check network latency
- Close other applications
- Reduce screen resolution

## Audio Issues

### No audio
- Check audio permissions
- Verify audio source selected
- Check volume levels
- Restart the app

### Echo
- Use headphones
- Disable audio on one side
- Check for audio loops

## Input Issues

### Mouse not working
- Check remote control permission granted
- Click on the remote window
- Restart the session

### Keyboard not working
- Check keyboard permission
- Try on-screen keyboard
- Check for shortcut conflicts

## File Transfer Issues

### Transfer failed
- Check file size limits
- Verify file type allowed
- Check disk space
- Try smaller files

### Slow transfer
- Check network speed
- Reduce file size
- Try ZIP compression

## Account Issues

### Can't log in
- Check email/password
- Reset password if needed
- Check email for verification
- Clear browser cache

### 2FA not working
- Check time sync on device
- Use backup codes
- Contact support to reset
