# Frequently Asked Questions

## General

### What is RemoteDesk?
RemoteDesk is a secure remote desktop application that lets you connect to and control devices remotely.

### Is there a free plan?
Yes, the free plan supports 1 device with basic features.

### What platforms are supported?
Windows, macOS, Linux, and web browsers.

## Connections

### How do I connect to a remote device?
Enter the 9-digit RemoteDesk ID and click Connect. The host must accept.

### Do both devices need RemoteDesk installed?
The host needs the app. Viewers can use the web app.

### Can I connect over the internet?
Yes, RemoteDesk works over any internet connection.

### How many people can connect simultaneously?
By default, one viewer per host. Enterprise supports multiple.

## Security

### Is my connection encrypted?
Yes, all connections use DTLS/SRTP end-to-end encryption.

### Can someone connect without my permission?
No, you must accept all incoming connections.

### Are sessions recorded?
Only if both parties consent. Recording is disabled by default.

### Where is my data stored?
Session data is not stored. Files are transferred P2P.

## Billing

### How do I upgrade my plan?
Go to Settings > Billing and select a new plan.

### Can I cancel anytime?
Yes, you can cancel at any time. Access continues until period end.

### Do you offer refunds?
Pro-rated refunds are available within 30 days.

### What payment methods are accepted?
Credit/debit cards. Enterprise supports invoicing.

## Technical

### What are the system requirements?
- Windows 10+, macOS 12+, Ubuntu 20.04+
- 4GB RAM minimum
- Internet connection

### Does it work behind firewalls?
Yes, with TURN relay support for restrictive networks.

### How much bandwidth is needed?
- Low quality: ~300 kbps
- Medium: ~1 Mbps
- High: ~3 Mbps

### Can I use RemoteDesk on mobile?
A web app is available. Native mobile apps are planned.
