# Quick Start Guide

## 1. Install (2 minutes)
Download and install RemoteDesk for your platform.

## 2. Sign In (1 minute)
- Open RemoteDesk
- Click "Sign In"
- Use your email or SSO

## 3. Get Your ID (instant)
Your 9-digit RemoteDesk ID is displayed on the main screen.

## 4. Connect (30 seconds)
- Ask for the remote device's ID
- Enter it and click Connect
- Wait for acceptance

## 5. Control (instant)
- Move your mouse to control the remote
- Use your keyboard normally
- Adjust quality if needed

## Next Steps
- Enable 2FA in Security settings
- Invite team members
- Set up your organization
- Explore advanced features

That's it! You're ready to use RemoteDesk.
