# Data Privacy Guide

## What Data We Collect
### Account Data
- Email address
- Name (optional)
- Organization info

### Usage Data
- Session metadata (duration, participants)
- Device info (OS, version)
- Error logs

## What We Don't Collect
- Screen content (never stored)
- Keystrokes (never logged)
- File contents (P2P transfer)
- Audio (real-time only)

## How We Use Data
- Service operation
- Security monitoring
- Support assistance
- Product improvement

## Your Rights
### GDPR (EU Users)
- Right to access
- Right to erasure
- Right to portability
- Right to object

### CCPA (California Users)
- Right to know
- Right to delete
- Right to opt-out
- Right to non-discrimination

## Data Retention
- Account data: Until deletion
- Session logs: 30-90 days (plan dependent)
- Audit logs: 1 year
- Deleted on account deletion

## Third Parties
- Stripe (payments)
- SendGrid (emails)
- Sentry (error tracking)
All GDPR-compliant with DPA.
