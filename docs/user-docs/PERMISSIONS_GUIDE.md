# Permissions Guide

## Host Controls
As the host, you control what the viewer can do:

### View
- **On**: Viewer sees your screen
- **Off**: Viewer sees a black screen

### Control
- **On**: Viewer can use mouse and keyboard
- **Off**: Viewer can only watch

### File Transfer
- **On**: Viewer can send files
- **Off**: No file transfers allowed

### Clipboard
- **On**: Clipboard sync enabled
- **Off**: Clipboard is isolated

### Recording
- **On**: Session can be recorded (with consent)
- **Off**: Recording disabled

## Changing Permissions
Click the shield icon in the session toolbar to adjust permissions.

## Permission Presets
- **Presentation**: View only
- **Collaboration**: View + Control
- **Full Access**: All permissions
- **Custom**: Your own combination

## Emergency Stop
Press Escape three times quickly to disable all remote input.
