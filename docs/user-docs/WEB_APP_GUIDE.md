# RemoteDesk Web App Guide

## Accessing the Web App
Navigate to https://app.remotedesk.io in your browser.

## Features
The web app supports:
- Viewing remote sessions
- Sending connection requests
- Managing your account
- Viewing session history
- Support tickets

## Limitations
The web app does not support:
- Hosting sessions (need desktop app)
- File transfers
- Clipboard sync
- Advanced settings

## Browser Requirements
- Chrome 90+
- Firefox 88+
- Safari 15+
- Edge 90+

## Getting Started
1. Sign in with your account
2. Enter a RemoteDesk ID to connect
3. Wait for the host to accept
4. View and control the remote session

## Tips
- Use Chrome for best performance
- Enable hardware acceleration
- Close unnecessary tabs
- Use a stable internet connection
