# RemoteDesk Desktop App Guide

## Installation
1. Download the installer from remotedesk.io/download
2. Run the installer for your platform
3. Launch RemoteDesk and sign in

## Getting Started
### Your RemoteDesk ID
- Each device has a unique 9-digit ID
- Share this ID so others can connect to you
- Found on the main screen

### Connecting to a Remote Device
1. Enter the 9-digit RemoteDesk ID
2. Click "Connect"
3. Wait for the host to accept
4. Start your remote session

### Screen Sharing
1. Click "Share Screen"
2. Select the screen or window to share
3. The viewer will see your selected source

## Controls
- **Disconnect**: Escape key or disconnect button
- **Fullscreen**: F11
- **Screenshot**: Ctrl+Shift+S
- **Quality**: Adjust in session toolbar

## File Transfer
1. Click the file transfer icon
2. Select files to send
3. Host must accept the transfer
4. Monitor progress in the transfer panel

## Permissions
During a session, the host controls:
- View access
- Remote control
- File transfer
- Clipboard sync
- Recording

## Troubleshooting
- Connection failed: Check both devices are online
- Black screen: Try changing quality settings
- Lag: Reduce quality or check network
- Audio issues: Check audio permissions

## Keyboard Shortcuts
| Key | Action |
|-----|--------|
| Escape | Disconnect |
| F11 | Fullscreen |
| Ctrl+Shift+S | Screenshot |
| Ctrl+Alt+Del | Send to remote (Windows) |
| Cmd+Option+Esc | Send to remote (Mac) |

## Security Notes
- Never share your RemoteDesk ID publicly
- Verify who is connecting before accepting
- Use strong passwords for your account
- Enable 2FA for additional security
