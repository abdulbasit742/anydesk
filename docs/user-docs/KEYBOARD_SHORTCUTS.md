# Keyboard Shortcuts Reference

## Session Controls
| Shortcut | Action |
|----------|--------|
| Escape | Disconnect (press 3x for emergency stop) |
| F11 | Toggle fullscreen |
| Ctrl+Shift+S | Take screenshot |
| Ctrl+Alt+F | Fit to window |
| Ctrl+Alt+1-4 | Switch quality (Low/ Med/ High/ Adaptive) |

## Remote Input
| Shortcut | Action |
|----------|--------|
| Ctrl+Alt+Del | Send to remote (Windows) |
| Cmd+Option+Esc | Send to remote (Mac) |
| Print Screen | Send to remote |
| Alt+Tab | Blocked (local only) |
| Cmd+Tab | Blocked (local only) |

## File Transfer
| Shortcut | Action |
|----------|--------|
| Ctrl+Shift+F | Open file transfer |
| Ctrl+Shift+A | Accept transfer |

## App
| Shortcut | Action |
|----------|--------|
| Ctrl+, | Settings |
| Ctrl+Shift+Q | Quit |
| Ctrl+N | New connection |

## Custom Shortcuts
You can customize shortcuts in Settings > Keyboard.
