# Admin Getting Started Guide

## Welcome, Admin!
As a RemoteDesk administrator, you manage users, devices, and policies for your organization.

## First Time Setup

### 1. Access Admin Dashboard
- Log in to RemoteDesk
- Click your profile -> "Admin Dashboard"
- Or go directly to https://app.remotedesk.io/admin

### 2. Review Organization Settings
- Organization name and logo
- Default language and timezone
- Data residency region
- Contact information

### 3. Configure Authentication
- Set password policy
- Enable mandatory 2FA
- Configure SSO (if enterprise)
- Set session timeout

### 4. Invite Users
- Go to Users -> Invite
- Upload CSV or enter emails
- Assign initial roles
- Send invitations

### 5. Set Policies
- Default session permissions
- Remote input rules
- File transfer settings
- Compliance requirements

## Daily Tasks
- Review pending invitations
- Check audit logs
- Monitor active sessions
- Respond to access requests

## Weekly Tasks
- Review security alerts
- Check license utilization
- Review device compliance
- Follow up on support tickets

## Monthly Tasks
- User access review
- Policy effectiveness check
- Generate usage report
- Review and update documentation
