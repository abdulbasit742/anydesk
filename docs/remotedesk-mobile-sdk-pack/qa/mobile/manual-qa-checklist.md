# Mobile Manual QA Checklist

## Onboarding
- [ ] App launch -> splash -> login
- [ ] Signup flow -> email validation
- [ ] Biometric prompt (if enabled)

## Dashboard
- [ ] Device list loads
- [ ] Search filters correctly
- [ ] Pull-to-refresh works
- [ ] Empty state shown

## Connect
- [ ] Enter RemoteDesk ID -> connect
- [ ] Invalid ID shows error
- [ ] Password prompt for protected device
- [ ] Recent devices pre-fill ID

## Session
- [ ] Remote video renders
- [ ] Status badge updates
- [ ] Toolbar shows on tap
- [ ] Fullscreen toggle
- [ ] Disconnect returns to dashboard

## Settings
- [ ] Biometric toggle persists
- [ ] Logout clears data
- [ ] Theme follows system

## Offline
- [ ] Queue actions offline
- [ ] Retry on reconnect
- [ ] Graceful error messages
