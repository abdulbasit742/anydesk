# Mobile App Permissions Checklist

- [ ] Camera (optional, for future video chat)
- [ ] Microphone (optional, for future audio)
- [ ] Local network (iOS NSLocalNetworkUsageDescription)
- [ ] Biometrics (Face ID / Touch ID / Fingerprint)
- [ ] Clipboard read (opt-in for sync)
- [ ] Keep awake during session
- [ ] Push notifications (for incoming requests)
- [ ] Photo library (for file transfer)
