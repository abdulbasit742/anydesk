# App Store Review Checklist

## iOS
- [ ] Provide demo account in review notes.
- [ ] Explain "background location" if used for Wake-on-LAN.
- [ ] Video demo of remote control flow.
- [ ] Privacy policy URL covers screen recording.
- [ ] No private API usage.

## Android
- [ ] Target SDK latest.
- [ ] Foreground service notification for persistent session.
- [ ] Data safety form: encrypted in transit, collected minimally.
- [ ] Explain accessibility service if used for remote input injection.
