# Security Settings

## Authentication
### Two-Factor Authentication (2FA)
1. Go to Settings > Security
2. Click **Enable 2FA**
3. Scan QR code with authenticator app
4. Enter verification code
5. Save backup codes

### Password Requirements
- Minimum 8 characters
- At least one uppercase letter
- At least one number
- At least one special character

## Session Security
### Require Approval
Always ask before allowing connections (enabled by default).

### Remote Control Permission
Control whether viewers can control your mouse/keyboard:
- **Always ask**: Prompt each time
- **Never**: View-only access
- **Allow for contacts**: Auto-allow trusted contacts

### Clipboard Sync
Control clipboard synchronization:
- **Enabled**: Sync clipboard both ways
- **Disabled**: No clipboard access

### File Transfer
Control file transfer:
- **Enabled**: Allow file transfers
- **Disabled**: Block file transfers

## Device Security
### Trusted Devices
View and manage devices that have accessed your account.
Remove devices you don't recognize.

### Session Timeout
Automatically end inactive sessions after a set time.

## Organization Security (Enterprise)
### IP Restrictions
Limit access to specific IP ranges.

### Session Recording
Record sessions for compliance (admin-configured).

### DLP Policies
Prevent transfer of sensitive file types.
