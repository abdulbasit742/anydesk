# Troubleshooting

## Connection Issues

### "Device Offline" Error
**Solution**:
- Ensure desktop app is running
- Check internet connection
- Restart desktop app
- Check firewall settings

### Connection Timeouts
**Solution**:
- Check both parties have stable internet
- Try different network (mobile hotspot)
- Disable VPN temporarily
- Check status page

### Black Screen
**Solution**:
- Host: Unlock screen if locked
- Host: Check correct display selected
- Update GPU drivers
- Reduce display resolution

## Performance Issues

### Laggy/Slow Connection
- Reduce quality setting to "Low"
- Close bandwidth-heavy applications
- Use wired connection instead of WiFi
- Check internet speed (need 1Mbps+)

### Low Frame Rate
- Reduce screen resolution
- Close unnecessary applications
- Check CPU usage on host
- Enable hardware acceleration

## Audio Issues
Note: Audio is disabled by default. Enable in settings if needed.

## Authentication Issues

### Forgot Password
1. Click "Forgot Password" on login page
2. Enter your email
3. Check inbox for reset link
4. Create new password

### Account Locked
Contact support@remotedesk.io if your account is locked.
