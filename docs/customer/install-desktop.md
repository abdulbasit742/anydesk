# Install Desktop Client

## Windows
1. Download from https://remotedesk.io/download/windows
2. Run `RemoteDesk-Setup.exe`
3. Follow installation wizard
4. Launch from Start Menu
5. Sign in with your account

### Windows Firewall
If prompted, allow RemoteDesk through Windows Firewall.

## macOS
1. Download from https://remotedesk.io/download/macos
2. Open `RemoteDesk.dmg`
3. Drag RemoteDesk to Applications
4. Launch from Applications folder
5. If "cannot be opened" appears: System Preferences > Security > Open Anyway

## Linux
### Ubuntu/Debian
```bash
wget https://remotedesk.io/download/remotedesk.deb
sudo dpkg -i remotedesk.deb
sudo apt-get install -f
```

### Fedora/RHEL
```bash
wget https://remotedesk.io/download/remotedesk.rpm
sudo rpm -i remotedesk.rpm
```

### AppImage
```bash
wget https://remotedesk.io/download/RemoteDesk.AppImage
chmod +x RemoteDesk.AppImage
./RemoteDesk.AppImage
```

## After Installation
1. Launch RemoteDesk
2. Sign in with your account
3. Your 9-digit RemoteDesk ID will be displayed
4. The app runs in the system tray

## Updates
Updates are automatic. You'll be notified when a new version is available.
