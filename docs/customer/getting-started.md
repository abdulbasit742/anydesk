# Getting Started with RemoteDesk

## What is RemoteDesk?
RemoteDesk is a secure, fast remote desktop solution for individuals and teams.

## Quick Start
1. Create an account at https://app.remotedesk.io
2. Download the desktop app
3. Get your 9-digit RemoteDesk ID
4. Share your ID to receive connections
5. Enter someone else's ID to connect

## System Requirements
### Desktop Client
- **Windows**: Windows 10 or later (64-bit)
- **macOS**: macOS 12 (Monterey) or later
- **Linux**: Ubuntu 20.04+ or Fedora 35+
- **RAM**: 4GB minimum
- **Internet**: 1 Mbps upload/download minimum

### Web Client
- Chrome 90+, Firefox 90+, Edge 90+, Safari 15+
- WebRTC support required

## Account Setup
1. Visit https://app.remotedesk.io/register
2. Enter your email and password
3. Verify your email address
4. Complete your profile
5. Download the desktop app

## Your First Connection
1. Open the desktop app - your 9-digit ID is displayed
2. Share this ID with your colleague
3. They enter it in their dashboard and click Connect
4. Accept the incoming request
5. Your screen is now shared!
