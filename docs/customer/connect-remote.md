# Connect to a Remote Computer

## Prerequisites
- Account at https://app.remotedesk.io
- The remote computer's 9-digit RemoteDesk ID
- Remote computer must be online

## From Web Dashboard
1. Log in to https://app.remotedesk.io
2. Go to Dashboard
3. Enter the 9-digit RemoteDesk ID
4. Click **Connect**
5. Wait for the host to accept

## From Desktop App
1. Open the desktop app
2. Click **Connect to Remote**
3. Enter the 9-digit RemoteDesk ID
4. Click **Connect**
5. Wait for the host to accept

## After Connection
- The remote screen appears in your browser/app
- Use your mouse and keyboard to interact (if permitted)
- Session toolbar provides additional controls

## Toolbar Controls
| Button | Function |
|--------|----------|
| Disconnect | End the session |
| Full Screen | Expand to full screen |
| Quality | Adjust stream quality |
| Chat | Open chat panel |
| File Transfer | Send files |
| Clipboard | Sync clipboard |

## If Connection Fails
- Verify the RemoteDesk ID is correct
- Ask the host to check if they're online
- Check your internet connection
- Try refreshing the page
- Contact support if persistent
