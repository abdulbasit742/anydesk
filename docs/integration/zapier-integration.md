# Zapier Integration

## Triggers
| Trigger | Description |
|---------|-------------|
| Session Started | When a remote session begins |
| Session Ended | When a remote session ends |
| Device Online | When a device comes online |
| Device Offline | When a device goes offline |
| New Payment | When a payment is received |

## Actions
| Action | Description |
|--------|-------------|
| Send Message | Send chat message in session |
| Create Session | Initiate a remote session |
| Update Device | Update device settings |

## Authentication
API Key authentication. Users generate API key in dashboard.

## Use Cases
- Log sessions to Google Sheets
- Send session notifications to email
- Create tickets in Zendesk for failed sessions
- Post to Slack when device goes offline
- Add payment info to accounting software
