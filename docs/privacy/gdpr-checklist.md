# GDPR Compliance Checklist

## Lawful Basis
- [ ] Consent obtained for analytics/tracking
- [ ] Contract basis for account data
- [ ] Legitimate interest for security logs
- [ ] Legal obligation for billing records

## Data Subject Rights
- [ ] Access request mechanism implemented
- [ ] Deletion request mechanism implemented
- [ ] Correction request mechanism implemented
- [ ] Portability request mechanism implemented
- [ ] Automated decision making: None

## Transparency
- [ ] Privacy policy published
- [ ] Cookie notice implemented
- [ ] Data processing info available
- [ ] Subprocessor list published

## Security
- [ ] Encryption at rest
- [ ] Encryption in transit (TLS 1.2+)
- [ ] Access controls implemented
- [ ] Audit logging enabled
- [ ] Breach notification procedure

## International Transfers
- [ ] EU data stays in EU (or SCCs in place)
- [ ] Standard Contractual Clauses signed
- [ ] Transfer Impact Assessment completed

## DPO
- [ ] DPO appointed
- [ ] Contact details published
- [ ] DPO involved in privacy decisions

## Documentation
- [ ] Records of processing activities
- [ ] DPIA completed for high-risk processing
- [ ] Regular compliance reviews scheduled
