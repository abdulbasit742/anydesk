# RemoteDesk Reseller Workflow Guide

## Reseller Process Map

### 1. Lead Registration
```
Reseller identifies prospect
  -> Registers deal in Partner Portal (72h protection)
  -> RemoteDesk approves/declines deal registration
  -> If approved: 90-day protection window
```

### 2. Sales Process
```
Reseller conducts discovery
  -> Schedules demo with RemoteDesk SE (optional)
  -> Proposes solution using approved pricing
  -> Customer signs order via Reseller
  -> Reseller submits order to RemoteDesk
```

### 3. Order Fulfillment
```
RemoteDesk validates order
  -> Provisions customer tenant
  -> Reseller receives confirmation
  -> Customer receives welcome email
  -> Reseller manages ongoing relationship
```

## Deal Registration Rules
- Must register before customer contacts RemoteDesk directly
- 90-day protection from approval date
- Extensions available with justification
- Dual-registration conflicts resolved by first registrant

## Commission Payment
- Monthly commissions paid net-30
- Minimum payout: $100
- Payment via ACH or PayPal
- Commission clawback if customer cancels within 90 days
