# RemoteDesk Partner Onboarding Guide

## Overview
Welcome to the RemoteDesk Partner Program. This guide walks you through onboarding as an authorized RemoteDesk partner.

## Partner Types
| Type | Description | Requirements |
|------|-------------|--------------|
| Referral | Refer customers, earn commission | Signed agreement |
| Reseller | Sell RemoteDesk subscriptions | Sales certification |
| Integration | Build integrations | Technical review |
| White-label | Rebrand RemoteDesk | Enterprise agreement |

## Onboarding Steps
1. **Sign Agreement** - Execute partner agreement via DocuSign
2. **Complete Training** - Finish partner certification (4 hours)
3. **Get API Keys** - Receive sandbox credentials
4. **Integration Test** - Validate against sandbox
5. **Go Live** - Production credentials issued

## Sandbox Access
- URL: `https://partner-sandbox.remotedesk.io`
- API Base: `https://partner-sandbox-api.remotedesk.io/v1`
- Webhook Tester: `https://partner-sandbox.remotedesk.io/webhooks`
