# RemoteDesk Troubleshooting Guide

## Quick Fixes

### App Won't Start
1. Restart your computer
2. Reinstall RemoteDesk
3. Check for OS updates
4. Temporarily disable antivirus
5. Contact support

### Cannot Connect
1. Check internet connection
2. Verify Desk ID is correct
3. Ask host to restart RemoteDesk
4. Try web app instead of desktop
5. Check firewall settings

### Black Screen
1. Check if host is sharing correct source
2. Try different capture source
3. Update graphics drivers
4. Disable hardware acceleration
5. Try lower quality setting

### Laggy Performance
1. Lower quality to "Low"
2. Close other applications
3. Use wired internet connection
4. Check bandwidth (speedtest.net)
5. Connect to closer server

### Audio Issues
1. Check system volume
2. Verify correct audio device selected
3. Update audio drivers
4. Restart RemoteDesk
5. Try without audio first

### File Transfer Fails
1. Check file size (< 2GB)
2. Verify file type allowed
3. Check disk space on receiving end
4. Try smaller file first
5. Restart session

### Clipboard Not Working
1. Verify permission granted
2. Try copying plain text only
3. Disable and re-enable clipboard
4. Clear clipboard (copy something else)
5. Restart session

## Diagnostic Information
When contacting support, provide:
- RemoteDesk version
- Operating system and version
- Browser (if using web app)
- Internet speed test results
- Error messages (screenshot)
- Steps to reproduce
