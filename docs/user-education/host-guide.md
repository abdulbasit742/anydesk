# RemoteDesk Host Guide

As a host, you control who can access your computer.

## Sharing Your Screen

### Starting a Session
1. Open RemoteDesk and log in
2. Your Desk ID is displayed prominently
3. Share your Desk ID with the viewer
4. When they request connection:
   - A notification appears
   - You see their name and Desk ID
   - Click "Accept" to allow
   - Click "Reject" to deny

### During the Session
- Green border indicates active sharing
- You maintain control of your computer
- The viewer can only do what you allow
- You can pause remote input anytime

### Permissions You Can Control
| Permission | What It Allows | Default |
|------------|---------------|---------|
| View Screen | See your desktop | On |
| Remote Input | Control mouse/keyboard | Off |
| File Transfer | Send/receive files | Off |
| Clipboard | Copy/paste between devices | Off |
| Chat | Text communication | On |

### Ending a Session
- Click "End Session" or disconnect button
- All connections are immediately closed
- Viewer no longer has access
- Session is logged for security

## Security Best Practices
- Only share Desk ID with trusted people
- Verify who is connecting before accepting
- Disable remote input if only viewing needed
- End session when no longer needed
- Lock your computer when away
- Review session history regularly
