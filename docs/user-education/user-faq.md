# RemoteDesk Frequently Asked Questions

## General

**Q: Is RemoteDesk free?**\nA: We offer a free tier with basic features. Premium and Enterprise plans offer advanced features.

**Q: What platforms are supported?**\nA: Windows 10+, macOS 11+, Ubuntu 20.04+, and modern web browsers.

**Q: Do I need to install anything?**\nA: The desktop app provides the best experience. The web app works without installation.

**Q: Can I use RemoteDesk on mobile?**\nA: Mobile support is on our roadmap. Currently desktop and web only.

## Technical

**Q: What internet speed do I need?**\nA: Minimum 1 Mbps for basic quality. 5+ Mbps recommended for HD.

**Q: Does RemoteDesk work behind firewalls?**\nA: Yes, using TURN servers for relay when direct connection is not possible.

**Q: Is my data encrypted?**\nA: Yes, all sessions use end-to-end encryption via WebRTC (DTLS + SRTP).

**Q: Can multiple people view my screen?**\nA: Yes, but each must be accepted individually. Enterprise plans support group sessions.

**Q: Does RemoteDesk work offline?**\nA: No, both parties need internet connectivity.

## Security

**Q: Can someone access my computer without permission?**\nA: No, all connections require explicit acceptance by the host.

**Q: Are sessions recorded?**\nA: Only if session recording is enabled (Enterprise feature, requires consent).

**Q: Who can see my Desk ID?**\nA: Only people you share it with. It is not publicly listed.

**Q: How do I report a security issue?**\nA: Email security@remotedesk.io. We respond within 24 hours.

## Billing

**Q: How do I cancel my subscription?**\nA: Go to Settings -> Billing -> Cancel Subscription.

**Q: Can I get a refund?**\nA: Contact billing@remotedesk.io within 30 days of charge.

**Q: How do I change my plan?**\nA: Go to Settings -> Billing -> Change Plan.
