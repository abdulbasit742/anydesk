# RemoteDesk Viewer Guide

As a viewer, you can connect to and (with permission) control remote computers.

## Connecting to a Host

### Request Connection
1. Get the host's 9-digit Desk ID
2. Enter it in the "Connect to" field
3. Click "Connect"
4. Wait for host to accept

### What You Can Do

#### View Only (Default)
- See host's screen in real-time
- Multiple quality settings available
- Full-screen mode (double-click)
- Zoom in/out

#### With Remote Input Permission
- Control host's mouse
- Type on host's keyboard
- Use keyboard shortcuts
- Note: Some system keys may be intercepted

#### With File Transfer Permission
- Send files to host
- Receive files from host
- Drag and drop support
- Progress indicator

#### With Clipboard Permission
- Copy from your device, paste on host
- Copy from host, paste on your device
- Supports text and images

#### Chat
- Send text messages during session
- Useful for communication without voice
- History visible during session

### Connection Quality
- Auto quality based on bandwidth
- Manual override in settings
- Lower quality = less bandwidth
- Stats panel shows FPS, latency, bitrate

## Tips for Better Experience
- Use keyboard shortcuts (Ctrl+Alt+F for full screen)
- Adjust quality if experiencing lag
- Use chat for coordination
- Request specific permissions as needed
- Be patient if host needs to approve actions
