# RemoteDesk Glossary

| Term | Definition |
|------|------------|
| Desk ID | Unique 9-digit identifier for each user |
| Host | The computer being accessed |
| Viewer | The person accessing the remote computer |
| Session | A single remote desktop connection |
| WebRTC | Technology enabling peer-to-peer connections |
| TURN | Server relay for NAT traversal |
| ICE | Protocol for establishing connections |
| SDP | Session Description Protocol |
| DTLS | Datagram Transport Layer Security |
| SRTP | Secure Real-time Transport Protocol |
