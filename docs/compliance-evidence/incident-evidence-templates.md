# Incident Evidence Templates

## Incident Report Template
```markdown
# Incident Report: INC-YYYY-NNNN

## Summary
- Date/Time: YYYY-MM-DD HH:MM UTC
- Severity: P1/P2/P3/P4
- Status: Open/Contained/Resolved/Closed
- Reporter: Name

## Timeline
- HH:MM - Detection method
- HH:MM - Initial response
- HH:MM - Containment actions
- HH:MM - Resolution

## Impact
- Users affected: N
- Services affected: List
- Data exposure: Yes/No/Unknown

## Root Cause
Description of root cause

## Remediation
- Immediate fix
- Long-term prevention

## Lessons Learned
- What went well
- What could improve

## Evidence
- [ ] Logs attached
- [ ] Screenshots saved
- [ ] Communications documented
```

## Post-Incident Review Template
```markdown
# Post-Incident Review: INC-YYYY-NNNN

## Attendees
- Name (Role)

## Timeline Review
Verified timeline of events

## Root Cause Analysis
5 Whys analysis

## Action Items
| Action | Owner | Due Date | Status |
|--------|-------|----------|--------|

## Sign-off
- [ ] Team Lead
- [ ] Security
- [ ] Engineering Manager
```
