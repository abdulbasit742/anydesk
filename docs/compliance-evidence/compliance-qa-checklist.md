# Compliance QA Checklist

## SOC 2 Readiness
- [ ] All CC6 controls implemented and tested
- [ ] All CC7 controls implemented and tested
- [ ] Access review completed this quarter
- [ ] Penetration test within 12 months
- [ ] Incident response plan tested
- [ ] Change management process documented
- [ ] Vendor risk assessments current
- [ ] Employee security training completed

## ISO 27001 Readiness
- [ ] Risk assessment completed
- [ ] Statement of Applicability approved
- [ ] Internal audit completed
- [ ] Management review conducted
- [ ] All required controls implemented
- [ ] Metrics collected and reviewed

## GDPR Readiness
- [ ] Privacy policy published
- [ ] Data processing agreements in place
- [ ] User data export tested
- [ ] User deletion process tested
- [ ] DPO appointed and contactable
- [ ] Data breach notification process tested
- [ ] Records of processing activities maintained
- [ ] DPIA completed for high-risk processing

## Evidence Review
- [ ] All evidence files current (within 90 days)
- [ ] Evidence covers all stated controls
- [ ] Evidence is tamper-evident
- [ ] Evidence retention meets requirements
