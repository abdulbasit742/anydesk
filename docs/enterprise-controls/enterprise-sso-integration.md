# RemoteDesk Enterprise SSO Integration

## Supported Providers
- SAML 2.0 (Okta, Azure AD, OneLogin)
- OIDC (Google Workspace, Auth0)
- LDAP/Active Directory

## Configuration Steps
1. Create application in IdP
2. Upload metadata to RemoteDesk
3. Map attributes (email, name, groups)
4. Test login flow
5. Enforce SSO

## Just-in-Time Provisioning
Users auto-created on first SSO login with mapped attributes.
