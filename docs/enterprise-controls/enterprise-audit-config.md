# RemoteDesk Enterprise Audit Configuration

## Audit Event Types
- user.login, user.logout, user.created
- session.started, session.ended
- policy.changed, permission.granted
- device.registered, device.trusted
- setting.changed

## Retention Policies
| Tier | Retention |
|------|-----------|
| Standard | 90 days |
| Enterprise | 1 year |
| Compliance | 7 years |
