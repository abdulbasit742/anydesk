# Enterprise Session Policies

| Policy | Options | Default |
|--------|---------|---------|
| Max session duration | 15min - 8hrs | 4hrs |
| Idle timeout | 5min - 60min | 15min |
| Require host approval | yes/no | yes |
| Allow unattended | yes/no | no |
| Record sessions | yes/no | no |
| Allow file transfer | yes/no/no-elevated | no |
| Allow clipboard | yes/no | no |
| Allow remote input | yes/no/view-only | yes |

Policies are evaluated at session start and cached for 5 minutes.
